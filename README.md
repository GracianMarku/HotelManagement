# Hotel Management System (Learning Project)

This is a learning project built using PHP and MySQL.

The purpose of this project was to understand:

- Basic CRUD operations
- Connecting PHP to MySQL
- Managing rooms and customers
- Handling reservations and payments
- Simple authentication (login system)

---

##  Tech Stack

- PHP
- MySQL
- HTML
- CSS
- Tailwind
- JavaScript

---

##  Features

- Room management (Create / Edit / Delete)
- Customer management
- Reservation system
- Payment handling
- Admin dashboard

---

## Learning Purpose

This project was built while learning backend fundamentals
